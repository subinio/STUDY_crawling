# 웹 크롤링


## 용어 정리
### 1. 크롤링(Crawling)
- 웹 상의 페이지를 수집해서 분류하고 저장한 후 나중에 쉽게 찾아볼 수 있도록 하는 역할을 하는 일종의 로봇
- 데이터를 저장한 후 쉽게 찾을 수 있도록 인덱싱
- 쉽게말해 수많은 데이터를 수집해서 나중에 꺼내보기 쉽도록 정리 해두는 것
- 스크래핑의 방법 중 하나

### 2. 파싱(parsing)
- 문장 혹은 문서를 구성 성분으로 분해하고 위계관계를 분석하여 문장의 구조를 결정하는 것
- 어떠한 웹페이지에서 내가 원하는 데이터를 특정 패턴이나 순서로 추출한 정보로 가공하는 것
- 일련의 정보를 분석하고 가공하는 것


### 3. 스크랩핑(scraping)
- 데이터를 수집하는 행위
- 데이터(자료)를 수집하는 모든 과정

<br>

```
<정리>

여러 곳에 흩어져 있는 수많은 자료를 스크래핑(크롤링)하여 수집된 데이터를
토대로 파싱(가공)하여 원하는 데이터를 빼내어 활용한다고 말할 수 있다.

통상 크롤링과 스크랩핑은 같은 의미로 쓰인다.
```

## 크롤링(crawling) 기법
1. HTML 페이지를 가져와서, HTML/CSS등을 파싱하고, 필요한 데이터만 추출하는 기법 *ex) Python BeautifulSoup*

2. Open API(Rest API)를 제공하는 서비스에 Open API를 호출해서, 받은 데이터 중 필요한 데이터만 추출하는 기법

3. 브라우저를 프로그래밍으로 조작해서, 필요한 데이터만 추출하는 기법 *ex) Python Selenium*
